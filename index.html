hat-drivers - Communication drivers#

For more information see:

  • online documentation

  • git repository

Runtime requirements#

  • python >=3.10

Install#

hat-drivers is available as PyPI project:

$ pip install hat-drivers

Build#

To install editable installation, together with python development dependencies, run:

$ pip install -e '.[dev]'

To install only python development dependencies, run:

$ pip install -r requirements.pip.txt

Build tool used for hat-drivers is pydoit. For listing available doit tasks, use:

$ doit list

Default task:

$ doit

creates wheel package inside build directory.
